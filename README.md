# Task 5 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset to identify patterns, trends, and relationships.

## Tools Used
- Google Colab
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Titanic Dataset

## EDA Techniques
- Dataset Information (.info())
- Summary Statistics (.describe())
- Missing Value Analysis
- Value Counts
- Histograms
- Boxplots
- Countplots
- Correlation Heatmap
- Pairplot

## Key Findings
- Most passengers were aged 20–40.
- Female passengers had higher survival rates.
- Passenger class influenced survival.
- Higher fares were associated with higher survival.
- Missing values were found in the Age column.
